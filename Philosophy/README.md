## 哲学六边形概要

### 六个核心维度

1. **认识论** - 保持怀疑，极度开放（破旧）
2. **科学哲学** - 模型思维，多维破局（立新）
3. **实用主义** - 快速行动，结果导向（落地）
4. **现象学** - 极致敏锐，洞察人心（用户）
5. **辩证法** - 预判趋势，把握时机（战略）
6. **伦理学/美学** - 坚守底线，追求卓越（灵魂）

### 核心洞察

1. **元范式**：哲学六边形是"道"（内功），具体创新范式是"术"（招式）
2. **动态平衡**：不同阶段需要不同的哲学权重组合
3. **知行合一**：哲学的价值不在知道，而在做到
4. **灰度思维**：实践者需要在矛盾中保持平衡

## 实践者的关键提醒

### 三个致命陷阱
1. **分析瘫痪** - 过度思考，行动迟缓
2. **沟通隔离** - 哲学语言脱离团队
3. **虚无主义** - 过度清醒消解动力

### 三个必要修正
1. **双重模式切换** - 战士模式(90%) + 将军模式(10%)
2. **黑箱封装** - 哲学作为底层代码，不写在UI上
3. **动态调权** - 根据生命周期调整哲学权重
